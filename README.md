# Customer Behavior Analysis
## Overview
This project focuses on analyzing customer behavior using Python, SQL, and Power BI to uncover meaningful business insights. The workflow includes data loading, cleaning, exploratory data analysis (EDA), SQL-based analysis, dashboard development, and report generation. The goal is to understand customer purchasing patterns, identify trends, and support data-driven decision-making.
## Dataset
The dataset contains customer-related information such as demographics, purchasing history, spending behavior, product categories, and customer ratings.
### Key Features
* Customer ID
* Age
* Gender
* Annual Income
* Purchase Amount
* Previous Purchases
* Product Category
* Review Rating
* Customer Segment
